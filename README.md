## Getting started

- [zig](https://ziglang.org/download/)
- [devkitPro](https://devkitpro.org/wiki/Getting_Started)

```
git clone https://github.com/zig-homebrew/zig-switch
cd zig-switch/
nix-build nix/devkitA64.nix
nix-shell
zig build
```
